# Evaluating Rock Position Estimation Pipeline Using 3D Sensors for Excavation Task

This repository hosts the source files for the project webpage associated with the M.Sc. thesis titled:

**Evaluating Rock Position Estimation Pipeline Using 3D Sensors for Excavation Task**

The webpage contains qualitative comparison media of the two evaluated segmentation models with the image data from the real excavator. Additionally, per-experiment breakdown videos are provided to showcase the proposed method for 3D rock position estimation.

🔴 **Webpage LIVE at:**<br>
[https://tau-alma.github.io/rock-3D-positioning-webpage](https://tau-alma.github.io/rock-3D-positioning-webpage)
